GitHub 部署方案
前置条件（需用户操作）：
1. 安装 gh CLI: brew install gh
2. 登录 GitHub: gh auth login
3. 配置 git: git config --global user.name "名字" / git config --global user.email "邮箱"
步骤命令说明
1. 初始化git init && git add -A && git commit -m "V4.0"在部署目录初始化
2. 创建仓库gh repo create shuzhi-workshop-v4 --public --source=. --push自动创建并推送
3. 启用 Pagesgh api repos/用户名/shuzhi-workshop-v4/pages -X POST -f source[branch]=main -f source[path]=/启用免费外网托管
4. 访问https://用户名.github.io/shuzhi-workshop-v4/外网URL，2-5分钟生效
